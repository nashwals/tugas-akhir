Tentang Sistem Ini (About This System)

Sistem ini menggunakan model statistik machine learning untuk memberikan estimasi tingkat stres berdasarkan berbagai faktor demografis, pekerjaan, dan kesehatan. Hasil estimasi bukan merupakan diagnosis klinis dan harus dikonsultasikan dengan profesional kesehatan mental untuk penilaian lebih lanjut.

This system uses statistical machine learning models to provide stress level estimations based on demographic, work-related, and health factors. Results are not clinical diagnoses and should be consulted with mental health professionals for further assessment.

Mode Estimasi: Isi form untuk mendapatkan estimasi skor stres berbasis model statistik. Data tidak akan disimpan (privacy-preserving).

Estimation Mode: Fill the form to get statistical stress score estimation. Data will NOT be saved (privacy-preserving).

Mode Kontribusi Penelitian: Selain mendapat estimasi, Anda akan menjawab 10 pertanyaan SOS-S untuk menghitung skor stres aktual. Data anonim akan disimpan dan dapat digunakan untuk pembaruan model melalui proses retraining terkontrol.

Research Contribution Mode: Besides estimation, you'll answer 10 SOS-S questions to calculate actual stress score. Anonymous data will be saved and can be used for model updates through controlled retraining processes.
